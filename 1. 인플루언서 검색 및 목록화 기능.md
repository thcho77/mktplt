## 기능개요
- 인플루언서 검색 요건을 세팅하고 각 플랫폼 (네이버블로그, Instagram, Facebook, Youtube, Thread, TikTok, X(Twitter), @cosme, douyin, xiaohongsu, bilibili)의 인플루언서를 서칭하고 목록화

## 검색 조건 
  - 계정 follower 수 (예:50,000~100,000)
  - 각 콘텐츠 View 수 (예:평균 30,000 View 이상)
  - 콘텐츠 특성 분류 : 커머스, 음악, 영화, 기술, 일상, 코미디, 먹방, 여행, Pet, 패션, 연예, 홈데코 등 (각플랫폼별 콘텐츠 분류)
  - 국가별 분류
  - 생물학적 분류 (예:성별, 연령대)
  - 팔로워의 특성 : 인구통계학적 특성 (남/여성, 연령대 등) - 가능한 경우, 전체 팔로워의 25% 이상의 공통 특성으로 분류

## 검색 프로세스
   1) 검색 대상 SNS플랫폼 선택 (체크박스로 복수 선택 가능)
   2) 검색 조건 선택
      - Follower 수 :           천부터            천까지 (선택 안할 경우 전체 대상)
      - 평균 콘텐츠 View 수(콘텐츠 평균 도달 수) : 검색 가능한 경우 적용
      - 콘텐츠 특성 분류 : 분류 가능한 경우 플랫폼의 콘텐츠 분류 기준을 가져와서 표시 (체크박스로 선택, 전체 카테고리도 선택하도록 지원)
      - 대상 국가명 : 드롭다운 메뉴로 선택
      - 생물학적 분류 (성별, 연령대)
      - 팔로워 특성별 분류
 3) 검색 결과
    - 아래 테이블 형식으로 데이터를 수집, 데이터베이스 구축
